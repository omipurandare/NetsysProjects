Description-
In this project a ProxyServer is implemented which acts as an intermediate in between the remote server and browser. It also supports caching. 

Folder structure-
PA-4_omkar_purandare-----|
			 |-----server(blocked_sites, host_ip_map, makefile,proxyserver.c,(cache folders))


Compiling code-
The server folder has the makefile, just run command "make" to get executables as "server"

Running code-
Server-
Give command "./server port_number timoout" on the command line to run the server to set port number and timeout, further no command can be given to server. Then run the browser or telnet to send request to server and observe the response

.
